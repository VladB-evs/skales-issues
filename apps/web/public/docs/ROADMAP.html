v3.5.0 — The Connections Update

X/Twitter integration, Safety Mode, and a wave of bug fixes

🐦
Communication · New Skill

X / Twitter Integration

Shipped in v3.5.0

Full OAuth 1.0a integration. Post tweets, read your timeline and @mentions, reply to tweets — directly from chat or via Telegram. Three permission modes: Send Only, Read & Write, Full Autonomous.

🛡️
System · Safety

Safety Mode (Handbrake)

Shipped in v3.5.0

Three-level shell command safety system. Safe (default) — blocks dangerous commands like rm -rf, format, fork bombs. Advanced — pauses and asks for Approve/Reject. Unrestricted — no blocking for power users.

📱
Mobile · Bug Fix

Mobile UI Fixes

Shipped in v3.5.0

Fixed mobile input bar disappearing behind keyboard, chat scroll-jump on new messages, hydration mismatch on mobile, and clipboard copy fallback for HTTP/Tailscale connections.

👁️
Vision · Bug Fix

OpenRouter Telegram Vision Fix

Shipped in v3.5.0

Telegram image uploads now work correctly when OpenRouter is the active provider. Skales auto-detects non-vision-capable models and falls back to GPT-4o Mini for image analysis.

Export / Import — Full Data Portability
Settings · Memory · Chat History
Users can currently not transfer their Skales setup between machines or back up their configuration. This feature adds a one-click Export that bundles all user data into a portable .skales-backup.zip — including memories, settings, persona config, skill definitions, and scheduled tasks. The matching Import wizard detects the file, validates it, and restores everything in the correct order, respecting foreign key constraints in the SQLite database.

Scope: Export/Import for Settings · Memories · Chat History · Persona · Skills · Cron Jobs
High Priority Data Portability SQLite Settings Memories

Skill AI — AI-Generated Custom Skills
Skills · Code Generation · No-Code
Let Skales write, debug, and optimize your custom skills. Describe what your skill should do — Skales generates the code, tests it, and installs it automatically. No dev setup required.

The user opens a chat-like interface and describes their skill in plain language: "When I say 'send morning briefing', compile a summary of my calendar + open tasks and send it via Telegram." Skales generates the handler, writes a manifest, runs a sandbox test, and adds the skill to the library — all without the user touching a file.
